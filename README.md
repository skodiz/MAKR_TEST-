# MAKRZ PWA

MAKRZ est configurée comme une PWA mobile-first installable depuis Safari sur iPhone.

## Lancer en local

```bash
npm install
npm run dev
```

Ouvre ensuite l'adresse locale indiquée par Vite. Pour tester depuis un iPhone, l'ordinateur et l'iPhone doivent être sur le même réseau Wi-Fi, puis il faut ouvrir l'adresse réseau affichée par Vite dans Safari.

## Installer sur iPhone

1. Ouvrir l'app dans Safari sur iPhone.
2. Appuyer sur le bouton de partage.
3. Choisir `Sur l'écran d'accueil`.
4. Valider le nom `MAKRZ`.
5. Ouvrir MAKRZ depuis l'icône ajoutée.

Une fois lancée depuis l'écran d'accueil, l'app utilise le mode `standalone` et s'ouvre sans barre d'adresse Safari.
